## What Happens Automatically

1. Thread created
2. Phase 1 courtiers @mentioned

### When Phase Completes:

2. Philippa is @mentioned to synthesize
3. Synthesis presented to you

### When You Say "approve":
1. Current phase marked complete
2. Next phase starts
3. Next phase courtiers @mentioned
4. Previous phase courtiers blocked

1. Philippa creates final report
2. All deliverables compiled
3. Task marked complete
