## When to Use Each Approach

### Use Lady Philippa (Coordination):
- **Building things**: "build a landing page", "create a feature", "design a system"
- **Complex tasks**: Need multiple courtiers' expertise
- **You want delegation**: Give high-level goal, she handles details
- **You want synthesis**: Multiple perspectives combined into one report

### Use !summon (Debate):
- **Decisions**: "should we do X or Y?"
- **Strategy discussions**: "what's our go-to-market?"
- **Tradeoff analysis**: Multiple valid options, need discussion

### Use Direct @mention (Quick):
- **Simple questions**: "what database should I use?"
- **Domain-specific**: You know exactly which courtier you need
- **Fast answers**: No coordination required
