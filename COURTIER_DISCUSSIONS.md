## How to Enable Great Discussions

### Let Them Talk

When Philippa coordinates a task, **let the courtiers discuss**. Don't jump in immediately. Watch them collaborate!

### Jump In When Needed

If they're stuck or going in the wrong direction:
```

```

They'll adjust immediately.

### Encourage Collaboration

When you want multiple perspectives:
```

```

Philippa will facilitate their discussion.

### Ask for Synthesis

When discussion is done:
```
@Lady Philippa what's the consensus?
```

She'll summarize their decisions.
